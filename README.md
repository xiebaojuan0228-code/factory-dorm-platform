# 制造中心宿舍管理平台

这是一个可直接发布到 GitHub Pages 的静态网页版本。

## 发布方式

1. 在 GitHub 新建一个仓库，例如 `factory-dorm-platform`。
2. 上传本文件夹内的所有文件到仓库根目录。
3. 打开仓库的 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 1 到 2 分钟，GitHub 会生成访问网址。

生成的网址通常类似：

`https://你的GitHub用户名.github.io/factory-dorm-platform/`
